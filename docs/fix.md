This is the procedure to run and confirm if the deploy is success:

1. Go to localhost:5173
2. Click on model tab at top right
3. Click on the top most trained CoLight model
4. Click on green deploy button
5. Press F5 to reset
6. Click on model tab at top right again
7. Now at bottom of that tab there is a blue View button. Click on view
8. Check if the simulation is running. If it show "Waiting for simulation data..." then this test is failed and the video still have not been loaded
9. If failed: Click on return symbol at top left. Then you're at Digital Twin Deploy now. Click Start then check if a video is loaded at Live Video feed below
